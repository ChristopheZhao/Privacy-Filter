# Code of Conduct

This project aims to be welcoming, practical, and respectful.

## Expected behavior

- be respectful and constructive
- assume good intent and focus on the work
- give clear technical feedback without personal attacks
- help keep discussions accessible to contributors with different experience levels

## Unacceptable behavior

- harassment, insults, or intimidation
- discriminatory language
- doxxing or sharing private information without consent
- deliberately disruptive behavior in issues, pull requests, or discussions

## Enforcement

Project maintainers may remove or moderate comments, issues, or contributions that make collaboration unsafe or unproductive.

## Reporting

If someone behaves in a way that makes collaboration unsafe, contact the maintainer privately through the repository owner's GitHub profile instead of escalating the situation in public.
